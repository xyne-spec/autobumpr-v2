# autobumpr-v2

A new and improved version of autobumpr.

# How to run

## edit values in Config.json

```json
{
  "accounts": [
    {
      "token": "DISCORD_USERBOT_TOKEN_HERE",
      "channelId": "CHANNEL_ID_HERE"
    }
  ]
}
```

## install dependencies
```bash
npm i
```

## execute file
```bash
node index.js
```
